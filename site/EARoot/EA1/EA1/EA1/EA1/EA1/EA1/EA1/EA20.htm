This class implements an automaton. #lt;br#gt; An automaton has a set of transitions, a set of states, an initial state and a set of final states. #lt;br#gt; The types of states, transitions, labels of transitions, are all generics and must extend the corresponding #lt;br#gt; super-class. #lt;br#gt; Each automaton object is ranked: it can represent either a single principal, or an ensemble of principals. #lt;br#gt; States and labels are tuples whose size equals the rank of the automaton. #lt;br#gt; <br/>@param #lt;S1#gt; the generic type in State#lt;S1#gt;, the content of a state.<br/>@param #lt;L1#gt; the generic type in Label#lt;L1#gt;, the content of a label.<br/>@param #lt;S#gt; the generic type of states<br/>@param #lt;T#gt; the generic type of transitions<br/>
