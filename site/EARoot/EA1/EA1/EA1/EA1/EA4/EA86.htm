Class implementing the projection function. #lt;br#gt; This function takes as arguments an automaton  (of rank#gt;1)  and an index, and returns the principal automaton (of rank 1) at position index.  #lt;br#gt; The projected automaton can store information, if needed, on the principals it was interacting with in the composition.  #lt;br#gt; In this case, the projected actions are addressed actions. #lt;br#gt;  The projection function is formally defined in Definition 5 of <br/><ul>
<li>Basile, D. et al., 2020. Controller synthesis of service contracts with variability. Science of Computer Programming, vol. 187, pp. 102344. (<a href="file://https://doi.org/10.1016/j.scico.2019.102344">https://doi.org/10.1016/j.scico.2019.102344</a>)</li></ul> <br/>@param #lt;S1#gt; the generic type of the content of states.<br/><br/>
